# toDo-ferrum

Это pet-проект созданный для повышения навыков программирования и изучения различных библиотек

---

## Основной стек:

- react
- redux/toolkit
- typescript
- material-ui
- formik
- yup

---

## Локальная разработка

- устанавливаем пакеты

```
npm i
```

- запускаем проект

```
npm run start
```

- в браузере заходим на localhost:3000
